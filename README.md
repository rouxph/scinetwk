# scinetwk

A scilab toolbox for graphs and networks. Metanet was part of scilab until version 4, dedicated to graph, then it became a external toolbox in scilab version 5.* ( https://atoms.scilab.org/toolboxes/metanet) but its developpement has been abandonned after.  The goal of this project is to make metanet aviable in modern version of scilab, conserving compatibility with the original functionnalities.

* retrieve original source code of metanet
* replaces functions using the old API or third party codes (in particular the show_graph/edit_graph module)
* rebuild the doculmentation
